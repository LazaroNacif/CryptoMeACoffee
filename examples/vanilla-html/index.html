>>> SUPER EASY SETUP

Just add one script tag to your website:

<script data-name="CMAC-Widget" src="widget.min.js" data-wallet="0x..." data-api="https://your-api.com/donate" data-creator-name="Your Name" data-color="#5F7FFF" data-position="Right"></script>

That's it! The widget auto-initializes and renders the floating button.

>>> MESSAGE SUPPORT

Supporters can now include an optional message (up to 500 characters) with their donations.

Messages are sent in the request body along with the amount, and your backend receives them automatically.

>>> CONFIGURATION OPTIONS

  • data-wallet - Your wallet address (required)
  • data-api - Your backend endpoint (required)
  • data-creator-name - Display name in modal
  • data-color - Button color (hex)
  • data-position - "Left" or "Right"
  • data-x_margin - Horizontal margin (px)
  • data-y_margin - Vertical margin (px)
  • data-network - "base-sepolia" or "base"
